# Roboarm #

### Apa itu Roboarm? ###

Roboarm merupakan sebuah robot yang dapat menulis di papan tulis dengan menggunakan Android dan Arduino sebagai penggeraknya.
Project ini di lakukan di IT dan Robotik Departemen SMAN 5 Surabaya.
